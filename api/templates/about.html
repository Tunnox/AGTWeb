About the AGT App

The AGT Web App is a secure digital platform developed for Amazing Grace Tabernacle, Sheffield. It helps members, leaders, and administrators collect, store, and access church records responsibly and transparently.

Through the app, users can connect, share safely, manage church activities efficiently, and ensure that all data is handled securely.

How to Use the App

Members can log in to update and manage personal information, register attendance, access sermons, submit prayer requests, and join community groups. Leaders and administrators can manage the database, schedule events, oversee giving records, and communicate securely with members.

Benefits

The AGT Web App strengthens connections, ensures GDPR compliance, and protects data with encryption and role-based controls. It builds trust through transparency and responsible stewardship of personal and financial data.

Our Policies

Built on data protection and transparency, the AGT Web App complies with UK GDPR. Members are informed about data use, and sensitive records are strictly role-based.

Best Practices

Use strong passwords, keep your details updated, never share login credentials, and report suspicious data. Communicate respectfully and with integrity in groups.

Data Protection & Security

All records are encrypted and backed up, with access controlled by roles. Regular audits ensure compliance, and rapid response procedures minimize risks in case of breaches.

AGT Web Application Policies

Data Protection Consent Access Accountability Security Best Practices

1. Data Protection & UK GDPR Compliance

Purpose: Ensure all data complies with UK GDPR.

  • Data collected and stored lawfully, fairly, and transparently.
  • Consent required before collecting personal info.
  • Data retained only as long as necessary.

2. Consent & Data Stewardship

  • No sharing without explicit consent.
  • Data used only for church purposes.
  • Consent can be withdrawn anytime.

3. Role-Based Access

  • Access granted strictly by role.
  • Admins = highest, Guests = least access.
  • No unauthorized credential sharing.

4. Accountability & Oversight

  • Supervised by AGT Admin Centre.
  • Regular audits ensure compliance.
  • Activity records maintained for accountability.

5. Security & Incident Response

  • All data encrypted and backed up.
  • Regular monitoring and audits.
  • Rapid response for errors/breaches.

6. User Best Practices

  • Strong, unique passwords required.
  • Keep personal details updated.
  • No credential sharing.
  • Respectful communication in groups.
